# Introduction

## Who can open the administrator panel ?

When using forumify for the firt time, the owner has the default Super Admin role and is the only one that can open the administrator panel. He can create as much roles as he want to, and give them access to the admin panel by clicking the "Administrator" box on the role creation page.

<Alert severity="info">
	Note : Moderators will not be granted an access to the administrator panel.
</Alert>

## Find the panel

When granted the administrator permissions, find the panel by clicking your name then "Admin". This will open a new tab.
