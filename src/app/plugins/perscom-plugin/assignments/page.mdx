# Assignments

## How to create an assignment record

To create an assignment record, go to your perscom users (Admin Panel > PERSCOM > Users) and click on the Quick Actions button and select new Assignment.

![assignment](/user-manual/plugins/perscom-plugin/assignment_record.jpg)
- **Users**: Here you can select an individual user or multiple.
- **Created At**: Date of record.
- **Type**: Whether it's a primary or secondary assignment.
- **Status**: The new status for the user, select "Keep current status" if you don't want to change it.
- **Specialty**: The new specialty for the user, select "Keep current specialty" if you don't want to change it.
- **Unit**: The user's new unit.
- **Position**: The user's position within their new unit.
- **Text**: If you want to supply a custom message which is displayed on the p-file.
- **Document**: Attach a document to the record.
