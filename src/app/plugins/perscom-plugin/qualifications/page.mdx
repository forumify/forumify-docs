# Qualifications

## How to create a qualification
Qualifications are a way for users to qualify on courses, enhancing their skills within the unit.

![Qualifications](/user-manual/plugins/perscom-plugin/qualification.jpg)
- **Name**: The name of the qualification.
- **Description**: Brief summary outlining the qualification, could include the requirements and rewards for it.
- **Image**: Visual insignia representing the qualification, displayed on qualification records.

See below for example of qualification insignia displayed on p-file records.
![Qualifications](/user-manual/plugins/perscom-plugin/qualification_display.jpg)

## How to assign a qualification to a user
To qualify users, go to your perscom users (Admin Panel > PERSCOM > Users) and click on the Quick Actions button and select New Qualification.

![promotion](/user-manual/plugins/perscom-plugin/promotion.jpg)
- **Users**: Here you can select an individual user or multiple.
- **Created At**: Date of record.
- **Type**: Whether it's a demotion or promotion.
- **Rank**: The user's new rank.
- **Text**: If you want to supply a custom message which is displayed on the p-file.
- **Document**: Attach a document to the record.
