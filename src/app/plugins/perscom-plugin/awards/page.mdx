# How to create an award
Awards are a way to recognize exceptional service, achievements which can incentivize participation and boost morale.

![Awards](/user-manual/plugins/perscom-plugin/award.jpg)
- **Name**: The name of the award.
- **Description**: Brief summary outlining the award, could include the requirements for it.
- **Image**: Visual ribbon or medal representing the award, displayed on p-files.

# How to award users
To award users, go to your perscom users (Admin Panel > PERSCOM > Users) and click on the Quick Actions button and select New Award.

![promotion](/user-manual/plugins/perscom-plugin/promotion.jpg)
- **Users**: Here you can select an individual user or multiple.
- **Created At**: Date of record.
- **Type**: Whether it's a demotion or promotion.
- **Rank**: The user's new rank.
- **Text**: If you want to supply a custom message which is displayed on the p-file.
- **Document**: Attach a document to the record.

<Alert severity="info">
    For demotions, follow the exact same steps, except select the type "Demote"
</Alert>
