# How to create a combat record
Combat Records are a great way to track user's attendance to missions, trainings, and meetings.
To create a combat record, go to your perscom users (Admin Panel > PERSCOM > Users) and click on the Quick Actions button and select New Combat Record.

![Combat](/user-manual/plugins/perscom-plugin/combat.jpg)
- **Users**: You can select individual or multiple users.
- **Created At**: Date of record.
- **Text**: The name of the combat record.
- **Document**: This lets you attach a document to the record.
