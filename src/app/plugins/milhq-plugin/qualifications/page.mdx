# Qualifications

Qualifications are a way for users to qualify on courses, enhancing their skills within the unit.

# How to create a qualification
 To create one, head over to Admin Panel > Organization > Qualifications and click on the + button.

![New Qualification](/user-manual/plugins/milhq-plugin/new-qualification.png)

- **Name**: Name of the qualification.
- **Description**: Explain what the qualification is, how to get it, what you get from receiving it.
- **Image**: Attach an image to the qualification.

# How to qualify a soldier

To qualify a soldier, head over to Admin Panel > Soldiers  and click on the Quick Action button and click New Qualification

![Qualify Soldier](/user-manual/plugins/milhq-plugin/qualify-soldier.png)

- **Soldiers**: You can attach an individual or multiple soldiers, making batch qualifying easier.
- **Created At**: Select the date of the qualification.
- **Qualification**: Select the qualification.
- **Text**: If you want to supply a custom message which is displayed on the p-file.
- **Document**: Attach a document to the qualification.
