# Card

The Card component is an enhanced version of the [box component](/guides/framework/components/box), by giving structure to the element.

<div class="flex gap-5 items-center">    
    <div class="w-80">
        ```html
            <div class="card">
                <div class="card-title">Card Title</div>
                <div class="card-body"> 
                    This is where content would go
                </div>
                <div class="card-footer">by forumify</div>
            </div>
        ```
    </div>
    <div class="card">
        <div class="card-title">Card Title</div>
        <div class="card-body"> 
            This is where content would go
        </div>

    </div>
