# Button
The button component adds styles for actions in forms, dialogs, and more.

<div className="flex gap-2">
    <button class="btn-primary">Primary</button>
    <button class="btn-cta">CTA</button>
    <button class="btn-outlined">Outlined</button>
    <button class="btn-link">Link</button>
</div>

```
<button class="btn-primary">Primary</button>
<button class="btn-cta">CTA</button>
<button class="btn-outlined">Outlined</button>
<button class="btn-link">Link</button>
```
the ``btn`` tag works on both ``<button>`` and ``<a>`` elements.

## Sizing
Buttons can come in two different sizes.

<div className="flex gap-2 items-center">
    <div className="text-center">
        <button class="btn-primary">Normal</button>

        ```
        <button class="btn-primary">Normal</button>
        ```
    </div>
    <div className="text-center">
        <button class="btn-primary btn-small mb-3">Small</button>

        ```
        <button class="btn-small">Small</button>
        ```
    </div>
</div>

## Icon
Sometimes buttons need to have an icon on them. This is where ``btn-icon`` comes into play by removing the margin.

    <button className="btn-primary btn-icon">Menu <CaretDown /></button>

        ```
        <button class="btn-primary btn-icon">Menu <i class="ph ph-caret-down"></i></button>
        ```
    <button className="btn-primary">Menu <CaretDown /></button>

        ```
        <button class="btn-primary">Menu <i class="ph ph-caret-down"></i></button>
        ```
